# Chatbot Voice AI

## Instalation
```
git clone https://github.com/fitri-hy/chatbot-voice-ai-python

# GUI Version
cd chatbot-voice-ai-python/gui
pip install pip install -r requirements.txt
python main.py

# CMD Version
cd chatbot-voice-ai-python/cmd
pip install pip install -r requirements.txt
python main.py
```

## Change Lang Voice
```
tts = gTTS(text=text, lang='id')
```
